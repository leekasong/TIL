# 커널 객체로 동기화하기 3

### 세마포어

* 시그널 : 자원계수가 0보다 클 때.
* 넌시그널 : 자원계수가 0일 때.
* 현재 자원의 계수는 최대 자원의 계수를 넘을 수 없다.
* 자원계수는 음수가 되지 않는다.

### 시그널 상태로 만들기
* 시그널로 만든다는 것은 자원계수를 0보다 큰 값으로 증가시킨다는 것이다.
* ReleaseSemaphore()를 사용한다.

### 넌시그널 상태로 만들기
* Mutex와 마찬가지로 WaitForXXX로 수행한다.
