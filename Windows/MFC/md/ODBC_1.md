# ODBC 실습하기

* 검색에 "ODBC 데이터 원본"을 타이핑한다.

![](../../images/ODBC_1/1.PNG)  

* MS ACCESS Database를 클릭하고 추가버튼을 누른다.

![](../../images/ODBC_1/2.PNG)  

* Microsoft Access Driver를 선택한다.

![](../../images/ODBC_1/3.PNG)  

* 디비 이름과 설명을 적는다.

![](../../images/ODBC_1/4.PNG)  

* mdb 파일이 있는 경로를 선택하면 왼쪽 리스트에 자동으로 파일이 뜨는데, 그것을 선택하고 확인을 누른다.
